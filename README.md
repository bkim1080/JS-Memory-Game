## Getting Started

To set up this project locally, follow these steps.

### Prerequisites

-   npm
    ```sh
    npm install npm@latest -g
    ```

### Installation

1. Clone the repository
    ```sh
    git clone https://github.com/bkim1080/JS-Memory-Game.git
    ```
2. Install NPM packages
    ```sh
    npm install
    ```
3. Run the "start" script
    ```sh
    npm start
    ```
